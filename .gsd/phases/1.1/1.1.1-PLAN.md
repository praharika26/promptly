# Plan 1.1.1: Typography and Theme Foundation

## Objective
Establish the "Space Grotesk" typography and create a unified Purple/Black color system using CSS variables and Tailwind configuration.

## Context
- .gsd/SPEC.md
- .gsd/DECISIONS.md
- app/layout.tsx
- app/globals.css

## Tasks

<task type="auto">
  <name>Install and Setup Space Grotesk</name>
  <files>
    - app/layout.tsx
  </files>
  <action>
    - Import 'Space_Grotesk' from 'next/font/google'.
    - Configure it with `variable: '--font-space-grotesk'` and `subsets: ['latin']`.
    - Apply the font variable to the `body` class in the RootLayout.
    - Remove unused Geist font imports to keep the bundle clean.
  </action>
  <verify>Check app/layout.tsx for Space_Grotesk import and usage.</verify>
  <done>Space Grotesk is correctly applied to the HTML body via a CSS variable.</done>
</task>

<task type="auto">
  <name>Implement Purple/Black Design Tokens</name>
  <files>
    - app/globals.css
  </files>
  <action>
    - Define CSS variables in `:root` for the new theme:
      - `--background`: Deep black (#050505).
      - `--primary`: Vibrant Purple (#a855f7).
      - `--primary-glow`: Glassy purple glow effect.
      - `--glass-border`: Translucent white/purple mix.
    - Update `@theme` (Tailwind 4 style) to map these variables.
    - Set default body color to `--background` and text to high-contrast white.
  </action>
  <verify>Run the app and observe the background color change to deep black.</verify>
  <done>CSS variables and Tailwind theme tokens are synchronized with the Purple/Black aesthetic.</done>
</task>

## Success Criteria
- [ ] Entire application uses 'Space Grotesk' as the primary font.
- [ ] Background is deep obsidian black.
- [ ] Primary purple tokens are available as Tailwind classes.
